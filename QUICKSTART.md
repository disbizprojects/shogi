## 🚀 Deploy to Vercel

### 1. Push to GitHub
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin your-github-repo
git push -u origin main
```

### 2. Deploy
- Go to https://vercel.com
- Click "Import Project"
- Select your GitHub repository
- Click "Deploy"

### 3. Done!
Your sites are live at:
```
https://your-vercel-project.vercel.app/enterprise-dashboard-preview
https://your-vercel-project.vercel.app/policy-configuration-preview
```
